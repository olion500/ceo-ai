# 10-Word Value Test

## What Is This Pattern?

If you can't explain your core value proposition in 10 words or less, your positioning isn't clear enough.

**Core principle:** Clarity drives conversion. Confusion drives bounce.

## How It Works

### The Test


### Why 10 Words?

**Short enough to:**

- Fit in elevator pitch
- Remember and repeat (word-of-mouth)
- Use in ad copy (character limits)
- Test with strangers (low attention span)

**Long enough to:**

- Convey actual value (not just category)
- Differentiate from competitors
- Include outcome (not just features)

## Why This Works

### 1. Clarity = Conversion

**Confused visitors:**

- Read value prop â†’ Don't understand â†’ Leave
- Conversion rate: 1-3%

**Clear visitors:**

- Read value prop â†’ Understand immediately â†’ Sign up
- Conversion rate: 5-15%

**5x difference from clarity alone**

### 2. Word-of-Mouth Growth

**Complex value prop:**

- User: "It's like... uh... you know how... it's hard to explain"
- Friend: "Sounds complicated, not interested"
- Viral coefficient: <1

**Simple value prop:**

- User: "Turns long videos into viral clips automatically"
- Friend: "Oh I need that, what's it called?"
- Viral coefficient: >1

### 3. Forces Product Focus

**If you can't explain it simply:**

- Product probably does too many things
- No clear core value
- Diluted positioning

**10-word test forces:**

- Identify ONE core value
- Remove unnecessary features from messaging
- Focus on outcome, not features

### 4. Testable with Strangers

**Friends/family:**

- Too polite to say "I don't get it"
- Fake understanding

**Strangers:**

- Brutal honesty
- If they don't get it, they say so

**Test with strangers = real validation**

## The 10-Word Value Test Process

### Step 1: Draft Initial Attempt (15 minutes)

Template options:

**Transformation template:**
> "Turn [input] into [output] [mechanism]"
- OpusClip: Turn long videos into viral clips automatically

**Problem-solution template:**
> "[Do thing] without [pain point]"
- Calendly: Schedule meetings without email back-and-forth

**Outcome template:**
> "[Desired outcome] for [who]"
- Stripe: Payments infrastructure for the internet

**Your draft:**
> "**_****_****_****_****_****_****_**___"

**Word count:** ___

### Step 2: Test with 10 Strangers (Day 1)

Find people who DON'T know your product:

- Coffee shop
- Subreddit
- Twitter DM
- Random Zoom call

**Script:**
"I'm testing a value prop. Can I read you one sentence and you tell me what you think the product does?"

[Read 10-word value prop]

"What do you think it does?"

**Pass criteria:**

- 7/10 people explain it correctly = PASS
- <7/10 = ITERATE

### Step 3: Iterate (Days 2-3)

**Common fixes:**

**If they don't understand the category:**

- Add category: "Workflow tool that..."
- Add context: "For content creators who..."

**If they confuse with competitor:**

- Add differentiator: "...without code"
- Add mechanism: "...automatically"

**If too vague:**

- Add specificity: "...for video podcasts"
- Add outcome: "...that go viral"

**Re-test after each iteration**

### Step 4: Validate with Metrics (Week 1)

**A/B test on landing page:**

Version A: Your 10-word value prop
Version B: Old value prop

**Measure:**

- Bounce rate (should drop)
- Time on page (should increase)
- Signup rate (should increase)

**If new version wins: You passed the test**

## Clarity Checklist

Does your 10-word value prop:

**Structure:**

- [ ] 10 words or less?
- [ ] Complete sentence?
- [ ] No jargon or buzzwords?
- [ ] Plain English (8th grade reading level)?

**Content:**

- [ ] States WHAT (not HOW)?
- [ ] Clear outcome/benefit?
- [ ] Specific enough to differentiate?
- [ ] Broad enough for target market?

**Validation:**

- [ ] Strangers understand immediately?
- [ ] Can be repeated from memory?
- [ ] Different from competitors?
- [ ] Makes people ask "how does it work?" (not "what does it do?")?

**If 10+ checkboxes: You have clarity**

## Failure Patterns to Avoid

### Anti-pattern #1: Thesis Statement

**Wrong:**
> "We believe content creation should be accessible to everyone"

**This is a MISSION, not a VALUE PROP**

### Anti-pattern #2: Category Only

**Wrong:**
> "The best video editing platform"

**Too generic, no differentiation**

### Anti-pattern #3: Aspirational Fluff

**Wrong:**
> "Empower creators to reach their full potential"

**Sounds nice, means nothing**

### Anti-pattern #4: Feature Soup

**Wrong:**
> "Video editing, clipping, publishing, and analytics all-in-one"

**Lists features, not outcome**

### Anti-pattern #5: Question Format

**Wrong:**
> "Want to create viral clips from your videos?"

**Doesn't state WHAT the product is**
