# Business Idea Evaluation: 외주 프리랜싱 (중개/아웃소싱 모델)

## Executive Summary

**One-line pitch:** 한국 고단가 시장에서 프로젝트를 수주하여 해외 저가 개발자에게 외주를 주고, 87-97% 마진을 확보하는 중개 비즈니스

**Target market:**
- 수주: 위시켓 한국 시장 (고단가)
- 외주: Freelancer.com 글로벌 시장 (저가)

**Business model:**
- 한국에서 수주 (예: 1,500만원) → 해외 외주 ($500/67만원) → 순수 마진 (1,433만원, 95.5%)
- PM + QA 역할로 품질 관리

**Unique angle:**
- 53배 가격 차이 활용한 차익 거래
- 글로벌 개발자 Pool 활용
- 시간 투자 최소화 (직접 개발 안 함)

## 수익성 비교

### 직접 개발 모델 (9개월)

```yaml
Phase 1 (0-3개월): 해외 경험
  - 월 수익: 100만원
  - 총 3개월: 300만원

Phase 2 (3-9개월): 한국 진입
  - 월 수익: 500-1,200만원
  - 총 6개월: 평균 850만원 × 6 = 5,100만원

9개월 총 수익: 5,400만원
월 평균: 600만원
```

### 중개 모델 (9개월)

```yaml
Phase 1 (0-3개월): 직접 개발 (평판 구축 필수)
  - 월 수익: 300-500만원
  - 총 3개월: 1,200만원

Phase 2 (3-6개월): 개발자 검증
  - 중개 시작 못 함 (검증 기간)
  - 월 수익: 500만원 (직접 개발 병행)
  - 총 3개월: 1,500만원

Phase 3 (6-9개월): 선택적 중개
  - 월 수익: 500-1,000만원 (리스크 관리)
  - 총 3개월: 2,250만원

9개월 총 수익: 4,950만원
월 평균: 550만원
```

**수익성 비교:**
- 직접 개발: 9개월 5,400만원 (월 평균 600만원)
- 중개 모델: 9개월 4,950만원 (월 평균 550만원)
- **차이: 직접 개발이 450만원 더 많음 (9개월 기준)**

**리스크 대비 수익:**
- 직접 개발: 낮은 리스크 + 높은 수익 = ✅✅ 최고
- 중개: 높은 리스크 + 비슷한 수익 = ⚠️⚠️ 위험

## 최종 추천: 단계별 접근

### ✅ 추천 전략: 하이브리드 접근

#### Phase 1 (0-6개월): 직접 개발로 시작
```yaml
목표:
  - 위시켓 5.0 평점 확보 (5개 프로젝트)
  - 월 500-1,200만원 수익
  - 포트폴리오 + 평판 구축
  - Freelancer 개발자 병행 테스트

why:
  - 리스크 낮음
  - 즉시 수익 가능
  - 평판 구축 (중개의 전제조건)
  - 개발자 검증 병행 가능
```

#### Phase 2 (6-12개월): 선택적 중개 검토
```yaml
조건 (모두 충족 시에만):
  - ✅ 위시켓 평점 5.0 (최소 5개 프로젝트)
  - ✅ 신뢰 개발자 5명 확보
  - ✅ 코드 리뷰 능력 검증
  - ✅ 품질 보증 시스템 구축
  - ✅ 법적 자문 완료

실행:
  - 표준화된 프로젝트만 중개 (ChatGPT, 스크래핑)
  - 복잡한 프로젝트는 직접 개발
  - 월 2-3개 중개 + 2-3개 직접

평가:
  - 재작업 발생률 < 15%?
  - 평점 유지 (5.0)?
  - 스트레스 관리 가능?
  → YES: 중개 확대
  → NO: 직접 개발 복귀
```

#### Phase 3 (12개월+): 최적 조합 찾기
```yaml
옵션 A: 하이브리드 유지
  - 내 강점: 직접 개발
  - 내 약점/시간 부족: 외주
  - 월 수익: 1,500-2,500만원

옵션 B: 직접 개발 집중
  - 고단가 프로젝트만
  - AI 도구로 효율화
  - 월 수익: 1,500-2,200만원

옵션 C: 에이전시 전환
  - 팀 구성 (PM, QA, 개발자)
  - 규모의 경제
  - 월 수익: 3,000-5,000만원
```

## 결론

### ⚠️ 중개 모델 단독 추천: NO

**이유:**
1. ⚠️⚠️⚠️ 품질 리스크 (20-30% 재작업)
2. ⚠️⚠️ 평판 리스크 (한 번 실패 = 게임 오버)
3. ⚠️ 법적/윤리적 리스크
4. ❌ 직접 개발보다 초기 수익 적음
5. ❌ 출시 시간 2배 (6개월 vs 3개월)

### ✅ 하이브리드 전략 추천: YES

**시작:**
1. **즉시:** 직접 개발 모델 시작
2. **6개월 후:** 조건 충족 시 선택적 중개 검토
3. **12개월 후:** 최적 조합 결정

**예상 결과 (12개월):**
- 직접 개발만: 월 1,500-2,200만원
- 하이브리드: 월 1,800-2,800만원
- 중개 전환: 월 1,000-1,500만원 (리스크↑)

### 💡 핵심 조언

**절대 이렇게 하세요:**
1. ✅ 직접 개발부터 시작 (즉시)
2. ✅ 위시켓 평점 5.0 확보 (필수)
3. ✅ 병행하며 개발자 검증
4. ✅ 6개월 후 조건부 중개

**절대 이러면 실패:**
1. ❌ 처음부터 중개 시작
2. ❌ 평판 없이 중개 시도
3. ❌ 개발자 검증 소홀
4. ❌ 복잡한 프로젝트 중개
