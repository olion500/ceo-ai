# 무제한 UI/UX 디자인 구독 서비스 (AI 보조)

## 한 줄 요약

Designfly 모델을 AI로 강화. 월 ₩990,000에 무제한 UI/UX 디자인 제공, ChatGPT + Midjourney로 빠른 배송.

## 비즈니스 모델

**타입:** Productized Service (AI 강화)
**가격:** ₩990,000/월 (무제한 요청)
**배송:** 1-2일/요청
**Capacity:** 월 1-2명 클라이언트 (초기)

## 실행 계획

### Week 1-2: 준비
```yaml
웹사이트:
- Framer/Webflow (노코드)
- 가격 투명하게 표시
- 포트폴리오 10개 (AI 생성)

AI 도구 준비:
- ChatGPT Plus ($20): 디자인 컨셉
- Midjourney Pro ($30): 비주얼
- Figma: 최종 디자인
```

### Week 3-4: 첫 클라이언트
```yaml
마케팅:
- Show HN 런칭
- LinkedIn 포스팅 (타겟: 스타트업 창업자)
- 한국 스타트업 커뮤니티

첫 프로젝트:
- 50% 할인 (₩495,000) 케이스 스터디용
- 빠른 배송 (24시간)
- 만족도 극대화 → 추천 유도
```

## 차별화

### 1. AI로 속도 10배
```
전통 디자이너: 1-2일/화면
AI 보조: 1-2시간/화면

프로세스:
1. ChatGPT: 와이어프레임 생성
2. Midjourney: 비주얼 컨셉
3. Figma: 최종 정리
```

### 2. 한국 스타트업 특화
```
타겟: Pre-Seed ~ Seed 스타트업
- 디자이너 채용 비용: ₩4,000,000/월
- 내 서비스: ₩990,000/월 (75% 절감)

포지셔닝: "스타트업 전용 디자인 구독"
```

### 3. 무제한 = 진입 장벽 낮음
```
심리: "한 달 써보고 판단" → 전환율 ↑
실제: 대부분 월 3-5개 요청 (관리 가능)
```

## 예상 수익

### Month 1
```
클라이언트 1-2명:
₩990,000 × 1-2 = ₩990,000 - 1,980,000

비용:
- AI 도구: $50/월 (₩67,000)
- 웹사이트: ₩0 (Framer 무료)

순수익: ₩923,000 - 1,913,000
```

### Month 3
```
클라이언트 5-8명:
₩990,000 × 6 (평균) = ₩5,940,000

비용:
- AI 도구: ₩67,000
- Capacity 도달 → 더 이상 수주 X

순수익: ₩5,873,000
```

### Month 6+
```
옵션 1: 가격 인상 (₩1,490,000)
옵션 2: 디자이너 채용 (Capacity ↑)
옵션 3: 대기자 명단 (Scarcity)
```

## 리스크

**AI 품질 한계**
- 완화: AI는 초안만, 최종은 수동 정리
- 완화: 무제한 수정 제공

**Capacity 부족**
- 완화: 월 5-8명 한정 (대기자 명단)
- 완화: 가격 인상으로 수요 조절
