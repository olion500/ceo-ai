# Business Idea Evaluation: 외주 프리랜싱 (직접 개발 모델)

## Executive Summary

**One-line pitch:** 해외 저가 시장에서 포트폴리오를 쌓은 후, 한국 고단가 시장으로 점프하여 월 300-600만원 수익을 창출하는 프리랜스 개발 비즈니스

**Target market:**
- Phase 1: Freelancer.com 글로벌 시장 (저가 경험 축적)
- Phase 2: 위시켓 한국 시장 (고단가 수익화)

**Business model:**
- 프로젝트 단위 수주 → 직접 개발 → 납품 → 대금 수령
- AI 도구(Claude Code, ChatGPT) 활용으로 생산성 2배 증가

**Unique angle:**
- 해외 경험을 한국 시장 차별화 요소로 활용
- AI 도구 마스터로 개발 속도 2배
- 53배 가격 차이를 활용한 단계적 수익 증대

## Detailed Evaluation

### 1. Problem Intensity: 7/10

**문제 정의:**
개발자들이 안정적이고 높은 수익을 올리기 어려운 문제

**Evidence:**
- ✅ 한국 개발자 평균 연봉: 4,000-6,000만원 (중급 기준)
- ✅ 프리랜서로 전환 시 불안정성: 수익 변동, 프로젝트 공백기
- ✅ 위시켓 프로젝트 존재: 62,940개 누적 = 지속적 수요
- ✅ 글로벌 시장: Freelancer 연 250만 프로젝트, Upwork 비슷한 규모
- ✅ 실제 데이터: 위시켓 평균 1,660만원, 지원자 9.4명 = 경쟁 가능

**Reasoning:**
- **7점인 이유:**
  - 문제는 실재하지만 "생존의 문제"는 아님 (직장이 대안)
  - 하지만 "더 많은 수익 + 시간 자유"는 강력한 동기
  - 프리랜서 시장 성장 추세 (노마드, 부업 트렌드)
- **8-9점 아닌 이유:**
  - Must-solve는 아님 (직장 다니면 해결됨)
  - 급박성 낮음 (천천히 준비 가능)

**Confidence Level:** High
**Key Assumptions:**
- 한국 고단가 시장 진입이 가능하다
- 해외 포트폴리오가 한국 시장에서 인정받는다

### 2. Market Size: 8/10

**Market Size Calculation:**

**TAM (Total Addressable Market):**
- 글로벌 프리랜스 시장: 연 $1.5조 (2025년 추정)
- 소프트웨어 개발 부문: 약 25% = $375B

**SAM (Serviceable Available Market):**
- 한국 + 영어권 시장 접근 가능
- 위시켓: 62,940개 프로젝트 누적
- Freelancer.com: 연 250만+ 프로젝트
- 개발 관련 추정: 50만 프로젝트/년
- 평균 프로젝트 단가: $5,000 (중간값)
- SAM = 50만 × $5,000 = **$2.5B (약 3.3조원)**

**SOM (Serviceable Obtainable Market - Year 1):**
- 현실적 목표: 월 2-5개 프로젝트
- 연간: 24-60개 프로젝트
- 평균 단가: $10,000 (한국 중심)
- SOM = 50개 × $10,000 = **$500,000 (약 6.7억원)**
- 시장 점유율: 0.02% (매우 작음)

**Reasoning:**
- **8점인 이유:**
  - 시장은 명확히 크고 성장 중
  - 진입 장벽이 낮아 접근 가능
  - 다양한 플랫폼 선택지 (위시켓, Freelancer, Upwork 등)
- **9-10점 아닌 이유:**
  - 개인이 접근 가능한 시장은 전체의 극히 일부
  - 1인 생산성 한계 존재

**Confidence Level:** High
**Key Assumptions:**
- 시장이 계속 성장한다
- AI 도구 확산으로 진입 장벽이 낮아진다

- ✅ 한국어 + 영어 가능 (양쪽 시장 접근)
- ✅ 해외 경험 포트폴리오 (한국 시장 차별화)
- ⚠️ 하지만 이것들이 독점 우위는 아님

**Reasoning:**
- **5점인 이유:**
  - 중간 수준의 경쟁
  - 진입 가능하지만 쉽지 않음
  - 차별화 전략 있으면 경쟁 가능
- **6-7점 아닌 이유:**
  - 경쟁자가 여전히 많음
  - AI 도구는 모두가 쓸 수 있음
  - 블루오션은 아님

**Confidence Level:** High
**Key Assumptions:**
- 전문 영역 특화 시 경쟁 낮아진다
- 품질과 속도로 차별화 가능하다

### 4. Monetization: 9/10

**Revenue Model:**
```yaml
Phase 1 (0-3개월): 해외 경험
  - 플랫폼: Freelancer.com
  - 프로젝트: 월 5개 × $150 = $750
  - 월 수익: 약 100만원 (실수령)

Phase 2 (3-12개월): 한국 진입
  - 플랫폼: 위시켓
  - 프로젝트: 월 3개 × 평균 500만원 = 1,500만원
  - 수수료 (10%): -150만원
  - 세금 (15%): -202만원
  - 월 실수령: 약 1,148만원

Phase 3 (12개월+): 안정화
  - 프로젝트: 월 2-3개 × 평균 800만원
  - 월 실수령: 약 1,500-2,200만원
```

**Pricing Evidence:**
- ✅ 위시켓 실제 데이터: 150만원~3,500만원
- ✅ 평균 1,660만원 (10개 프로젝트 분석)
- ✅ 결제 시스템 명확: 플랫폼 에스크로
- ✅ 고객 지불 의향: 검증됨 (62,940개 프로젝트)

**Unit Economics:**
```yaml
한국 프로젝트 (평균):
  수주가: 1,500만원
  작업 시간: 60시간 (AI 도구 활용)
  시간당 단가: 250,000원

  비용:
    - 플랫폼 수수료: -150만원 (10%)
    - 세금: -202만원 (15%)
    - 순수익: 1,148만원

  시간당 순수익: 191,333원
```

**Reasoning:**
- **9점인 이유:**
  - 수익 모델 극도로 명확
  - 결제 시스템 검증됨
  - 단가 데이터 풍부
  - 즉시 수익 발생 가능
- **10점 아닌 이유:**
  - 시간 판매 모델 = 확장성 제한
  - 프로젝트 공백기 리스크

**Confidence Level:** High
**Key Assumptions:**
- 위시켓에서 월 2-3개 수주 가능
- 가격 협상 여지 존재

### 5. Differentiation: 6/10

**Differentiation Strategy:**

**1. AI 도구 마스터:**
- Claude Code로 프로젝트 전체 생성
- 개발 시간 50% 단축
- ⚠️ 하지만 다른 개발자도 곧 따라올 수 있음

**2. 해외 경험 포트폴리오:**
- Freelancer 50+ 프로젝트 완료 실적
- 영어 커뮤니케이션 능력
- 글로벌 스탠다드 코드 품질
- ⚠️ 차별화 요소이지만 독점적이진 않음

**3. 속도:**
- 빠른 납품 (기존 대비 2배)
- 고객 만족도 증가
- 재주문율 향상

**4. 틈새 전문화:**
- 특정 스택 전문가 (예: React + Nest.js + AI 통합)
- 블록체인, 특수 API 등

**Competitive Positioning:**
```
일반 개발자: 범용, 느림, 고가
나: AI 활용, 빠름, 합리적 가격
초보자: 저가, 품질 낮음
```

**Reasoning:**
- **6점인 이유:**
  - 차별화 요소는 있지만 독점적이진 않음
  - AI 도구는 누구나 쓸 수 있음
  - 속도와 품질의 조합이 차별화
- **7-8점 아닌 이유:**
  - 패러다임 전환 수준은 아님
  - 따라하기 어렵지 않음
  - 지속 가능성 불확실

**Confidence Level:** Medium
**Key Assumptions:**
- AI 도구 활용 격차가 당분간 유지됨
- 속도가 실제 차별화 요소가 됨

### 6. Time to Market: 8/10

**Launch Timeline:**

**Week 1-4: 준비 (1개월)**
```yaml
- Freelancer 계정 생성 및 프로필 작성
- 위시켓 전문가 등록
- 샘플 포트폴리오 3개 제작
- Claude Code Pro 구독
```

**Month 2-3: 해외 경험 (2개월)**
```yaml
- Freelancer에서 소형 프로젝트 5-10개
- $100-200 범위
- 5.0 평점 확보
- 총 투자: 시간 (주 15-20시간)
```

**Month 4: 한국 진입 (1개월)**
```yaml
- 위시켓 첫 프로젝트 수주
- 150-300만원 범위
- 직접 개발 완료
- 5.0 평점 시작
```

**Total Time to Revenue: 3-4개월**
- 해외 시장: 즉시 수익 (소액)
- 한국 시장: 3-4개월 후 본격 수익

**Reasoning:**
- **8점인 이유:**
  - 1개월 이내 첫 수익 가능 (해외)
  - 3개월 이내 한국 진입 가능
  - 특별한 인프라 불필요 (노트북만 있으면 됨)
  - 점진적 시작 가능 (부업으로 시작)
- **9-10점 아닌 이유:**
  - 한국 고단가 수익까지는 3개월 필요
  - 평판 구축 시간 필요

**Confidence Level:** High
**Key Assumptions:**
- 포트폴리오 3개를 1개월 내 제작 가능
- 해외 첫 프로젝트를 2개월 내 수주 가능

- 프로젝트 수주: 혼자

```yaml

고객 소통: 10시간
총: 150시간/월 (주 37.5시간)
```

**확장 가능성:**
```yaml
1인: 월 2-5개 프로젝트
  → 수익: 1,000-2,500만원

2인 (파트너): 월 6-12개
  → 수익: 3,000-6,000만원

3-5인 (팀): 월 15-30개
```

**필요 스킬:**
- 개발 (핵심): 80%
- PM/영업: 15%
- 디자인: 5% (외주 or 템플릿)

**Reasoning:**
- **9점인 이유:**
  - 완벽한 1인 비즈니스 모델
  - 시간과 장소 자유로움
  - 최소 인프라 (노트북만)
  - 고정비 거의 없음 (플랫폼 수수료만)
- **10점 아닌 이유:**
  - 확장성에 한계 (시간 판매)
  - 휴가 = 수입 0
  - 아플 때 리스크

**Confidence Level:** High
**Key Assumptions:**
- AI 도구로 1인 생산성 충분
