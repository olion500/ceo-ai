# Next.js AI SaaS Boilerplate (한국어 특화)

## 한 줄 요약

ShipFast 모델의 한국어 버전. Next.js 14 + Supabase + OpenAI API + 한국 결제(Toss) 보일러플레이트.

## 비즈니스 모델

**타입:** One-Time Purchase Developer Tool
**가격:** ₩199,000 (ShipFast: $200-300)
**타겟:** 한국 개발자 (AI SaaS 만들고 싶은 사람)
**수익:** 판매당 순수익 (Gumroad 수수료 10%)

## 검증된 수요

```
ShipFast 실적:
- Month 1: $36,400
- Lifetime: $1,200,000+

한국 시장 공백:
- ShipFast: 영문만
- 한국 개발자: ChatGPT 붐 이후 AI SaaS 관심 ↑
- 기회: 한국어 + 한국 인프라 특화
```

## 포함 기능

### Core Stack
```typescript
- Next.js 14 (App Router)
- TypeScript (strict mode)
- Supabase (Auth + DB + Storage)
- Prisma (ORM, 선택)
- Tailwind CSS + shadcn/ui
- OpenAI API (ChatGPT 통합)
```

### 한국 특화
```typescript
- Toss Payments (토스페이먼츠 통합)
- 카카오페이, 네이버페이 옵션
- 한국어 i18n (기본)
- AWS Seoul Region 최적화
- Naver Cloud 옵션
```

### 기능
```typescript
- Authentication (이메일, 소셜)
- Subscription (월/연 결제)
- Admin Dashboard
- OpenAI API Streaming
- SEO 최적화 (한국어)
- 이메일 템플릿 (한국어)
```
