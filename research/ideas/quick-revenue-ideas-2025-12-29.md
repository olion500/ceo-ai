## 🇰🇷 한국 시장 특화 아이디어 (5개)

### 1. 프리랜서 단가 투명성 플랫폼 "공개단가"

**문제점:**

- 한국 프리랜서 개발자/디자이너 단가 정보가 파편화되고 불투명함
- 온라인 정보는 개인 경험 위주로 프로젝트 유형/계약 구조 미반영
- 클라이언트는 적정 단가를 모르고, 프리랜서는 협상력 부족

**솔루션:**

- **익명 단가 데이터베이스**: 프리랜서가 실제 계약 정보 (기술스택/연차/단가/프로젝트 유형) 익명 제출
- **AI 단가 추천**: 프로젝트 요구사항 입력 → 적정 단가 범위 제시
- **시장 인사이트**: 기술별/경력별 단가 트렌드, 수요 분석

**수익 모델:**

- 프리미엄 구독 (프리랜서용): ₩29,000/월 - 상세 단가 데이터, 협상 팁, 계약서 템플릿
- 기업용 플랜: ₩99,000/월 - 무제한 견적, 시장 리포트, 채용 지원

**빌드 시간:** 1-2주 (간단한 폼 + 데이터베이스 + AI 분석)

**검증 방법:**

1. OKKY, 이랜서 커뮤니티에 설문 게시 ("단가 정보 불투명 문제 공감하시나요?")
2. 100명 단가 데이터 수집 후 무료 베타 공개
3. 프리미엄 기능 사전 판매 ($19 early bird)

**성공 확률:** ⭐⭐⭐⭐ (4/5)

- ✅ 명확한 pain point (단가 정보 불투명성)
- ✅ 타겟 명확 (한국 프리랜서 + 고용 기업)
- ✅ 즉시 가치 제공 (첫 사용부터 단가 정보 획득)
- ⚠️ 데이터 수집 초기 cold start 문제

**참고 사례:**

- Levels.fyi (미국 테크 연봉 투명성)
- Bank Statement Converter (니치 도구, $38K MRR)

**솔루션:**

- **계약서 업로드**: PDF/DOCX 계약서 크롬에서 바로 열기
- **AI 자동 분석**:
  - 불리한 조항 하이라이트 (책임 범위, 지급 조건, 납품 기한)
  - 리스크 점수 (1-10점)
  - 수정 제안 텍스트 제공
- **템플릿 라이브러리**: 검증된 계약서 템플릿 제공

**수익 모델:**

- 무료: 월 2회 분석
- 프리미엄: ₩19,000/월 - 무제한 분석, 템플릿, 변호사 Q&A
- 기업: ₩99,000/월 - 팀 계정, 계약 히스토리 관리

**기술 스택:**

2. 반복되는 질문 → AI 자동화
3. Chrome Web Store 무료 버전 출시 → 프리미엄 전환율 측정

**성공 확률:** ⭐⭐⭐⭐⭐ (5/5)

### 3. 인스타그램 공동구매 자동화 SaaS "공구매니저"

- 인스타 공동구매 진행자가 수작업으로 주문 관리 (DM 확인, 입금 추적, 배송 안내)

- **입금 확인 자동화**: 은행 입출금 내역 연동 → 주문자 자동 매칭
- **배송 추적**: 택배사 API 연동 → 자동 발송 알림
- **정산 리포트**: 수익/비용 자동 계산

**수익 모델:**

- 무료: 월 거래 50건까지
- 스타터: ₩29,000/월 - 월 200건, DM 자동화
- 프로: ₩79,000/월 - 무제한, 은행 연동, 정산 리포트

**빌드 시간:** 1주 (인스타 API + 은행 스크래핑 + 알림톡)

**검증 방법:**

1. 인스타 공구 진행자 5명 찾기 (해시태그 #공동구매)
2. 무료로 도구 제공 → 피드백 수집
3. 유료 전환 제안 (성공 시 월 ₩29K)

**성공 확률:** ⭐⭐⭐⭐ (4/5)

- ✅ 명확한 타겟 (인스타 공구 진행자)
- ✅ 반복 구매 (매달 공구 진행)
- ✅ 저경쟁 니치 (기존 도구 없음)
- ⚠️ 인스타 API 정책 변경 리스크

**참고 사례:**

- 인스타 공동구매 성공 사례 (연구 자료)

### 4. AI 퇴사 준비 코치 "퇴사메이트"

**문제점:**

- 한국 직장인들 퇴사/이직 고민 많지만 누구에게 상담하기 어려움
- 퇴직금 계산, 연차 정산, 경력기술서 작성 등 실무 정보 부족
- 감정적 결정으로 후회하거나, 준비 부족으로 손해

**솔루션:**

- **퇴사 타이밍 진단**: 현재 상황 입력 → AI가 퇴사 적기 판단 (재무/경력/건강 종합)
- **퇴사 체크리스트**: 법적 권리 (퇴직금/연차수당), 준비사항 (건강보험/실업급여)
- **경력기술서 AI 작성**: 직무 경험 입력 → 매력적인 경력기술서 자동 생성
- **익명 커뮤니티**: 퇴사 고민 나누기 (동종업계 연결)

**수익 모델:**

- 무료: 기본 진단 + 체크리스트
- 프리미엄: ₩49,000 (일회성) - AI 경력기술서 무제한, 1:1 코칭 채팅, 퇴사 타이밍 시뮬레이션

**빌드 시간:** 5-7일 (랜딩 페이지 + GPT-4 프롬프트 + 결제)

**검증 방법:**

1. 블라인드 앱에 "퇴사 고민 설문" 게시
2. 결과 보고 싶으면 이메일 입력 → 리드 수집
3. 무료 리포트 보낸 후 프리미엄 제안

**성공 확률:** ⭐⭐⭐⭐ (4/5)

- ✅ 강력한 pain point (퇴사 준비 스트레스)
- ✅ 일회성 고가 판매 가능 (₩49K)
- ✅ 바이럴 가능성 (블라인드 입소문)
- ⚠️ 시즌성 (연말/상반기 수요 집중)

**참고 사례:**

- Resume.io (이력서 빌더)
- AI 콘텐츠 도구 (주말 빌드 → ₩3K+ 매출)

### 6. Chrome Extension: Developer Error Logger "ErrorSnap"

**Problem:**

- Developers constantly screenshot errors to share with team/Stack Overflow
- Manual process: screenshot → annotate → upload → share
- Context often lost (console logs, network tab)

**Solution:**

- **One-Click Error Capture**: Detect console errors → auto-capture screenshot + logs + network
- **Instant Shareable Link**: Upload to cloud → generate link with full context
- **Team Library**: Save common errors + solutions in searchable database
- **AI Suggestions**: Analyze error → suggest fixes from Stack Overflow/docs

**Revenue Model:**

- Free: 10 captures/month
- Pro: $9/month - Unlimited captures, team library, AI suggestions
- Team: $29/month - Shared workspace, analytics

**Build Time:** 3-5 days (Chrome extension + cloud storage + OpenAI API)

**Tech Stack:**

- Manifest V3 Chrome Extension (Cursor AI)
- Cloudflare R2 (image storage)
- Supabase (database)
- GPT-4 API (error analysis)

**Validation:**

1. Post on /r/webdev: "Would you pay $9/month for instant error sharing?"
2. Build free version → track conversion to paid
3. Launch on Product Hunt

**Success Probability:** ⭐⭐⭐⭐⭐ (5/5)

- ✅ Developer tools = premium pricing justified
- ✅ Chrome extension avg $862K/year revenue
- ✅ Daily use case (developers debug daily)
- ✅ Network effect (teams adopt together)

**Similar Success:**

- Bluedot ($1.5K MRR, 50% monthly growth)
- Developer tools command $49-99/month easily

- Stripe sends receipts to email → buried in inbox
- Tax time = nightmare (finding all receipts)

**Solution:**

- **Email Integration**: Forward Stripe receipts to custom email
- **Auto-Categorization**: AI categorizes expense (software, equipment, travel)
- **PDF Report Generation**: One-click generate IRS-compliant expense report
- **Multi-Currency**: Auto-convert to home currency
- **Tax Deduction Suggestions**: AI highlights tax-deductible items

**Revenue Model:**

- Free: 20 receipts/month
- Freelancer: $9/month - Unlimited receipts, auto-categorization
- Business: $29/month - Team accounts, accountant access, QuickBooks export

**Build Time:** 5-7 days (email parsing + AI categorization + PDF generation)

**Validation:**

1. Post on Indie Hackers: "Freelancers, how do you track Stripe expenses?"
2. Offer manual service (email receipts → I send report)
3. 10 manual customers → automate → paid product

## 실패 방지 가이드

### 🚫 하지 말아야 할 것

1. **완벽주의**: 2주 넘게 빌드하지 마세요 → 출시 후 개선
2. **새 기술 배우기**: 아는 스택으로만 빌드 (속도가 생명)
3. **모든 기능 넣기**: 핵심 1개만 → 나머지는 유료 고객 요청 후
4. **무료로 시작**: Day 1부터 유료 → 진지한 고객 필터링
5. **대중 타겟**: "모두를 위한" → "특정 니치만"
6. **경쟁자 무시**: 경쟁자 0개 = 수요 없음 (3-10개가 ideal)

### ✅ 반드시 해야 할 것

1. **문제부터 검증**: 빌드 전 10명과 인터뷰 → 진짜 문제인지 확인
2. **수동 MVP**: 자동화 전에 수동으로 5명에게 서비스 → 핵심 파악
3. **사전 판매**: 빌드 전에 5명에게 팔기 → 진짜 수요 검증
4. **즉시 런칭**: 버그 있어도 2주 내 출시 → 피드백 수집
5. **매일 마케팅**: 개발 시간 50% = 마케팅 시간 50%
6. **가격 올리기**: 가치 증가하면 주저 없이 가격 인상
