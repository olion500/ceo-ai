# 웹 스크래핑 자동화 서비스 (위시켓 → Upwork 중개)

## 한 줄 요약

한국(위시켓)에서 웹 스크래핑 프로젝트를 수주하고, 해외(Upwork) 개발자에게 외주하여 90%+ 마진 확보.

## 비즈니스 모델

**타입:** Service Arbitrage (서비스 차익 거래)
**플랫폼:** 위시켓 (수주) + Upwork (외주)
**마진율:** 90-95%
**수익 구조:** (한국 수주가 - 해외 외주비) = 순수 마진

## 왜 이 아이디어인가?

### 검증된 가격 차이
```
실제 데이터 (2025-12-29 분석):

한국 시장 (위시켓):
- 간단한 웹 스크래핑: ₩3,000,000 - 5,000,000
- 복잡한 스크래핑: ₩10,000,000 - 20,000,000

해외 시장 (Upwork):
- 간단한 스크래핑: $100 - 300 (₩130,000 - 400,000)
- 복잡한 스크래핑: $500 - 1,000 (₩670,000 - 1,340,000)

마진: ₩2,600,000 - 18,660,000 (90-95%)
```

### 지속적 수요
- **위시켓:** 웹 스크래핑 프로젝트 월 10-20개 게시
- **숨고:** 매크로/프로그램 개발 수요 높음
- **Upwork:** 스크래핑 전문가 수천 명 대기

### 기술 표준화
- 웹 스크래핑은 전 세계 동일 기술 (Python, Beautiful Soup, Selenium)
- 언어 장벽 낮음 (코드는 영어 기반)

## 실행 계획 (4주)

### Week 1-2: 준비 및 프로필 구축
```yaml
Week 1: 플랫폼 설정
  Day 1-3: 위시켓 프로필 생성
    - 기술 스택 강조:
      * Python (Beautiful Soup, Scrapy, Selenium)
      * 데이터 처리 (Pandas, Excel)
      * API 연동 (REST API)
    - 포트폴리오 준비:
      * GitHub 샘플 프로젝트 3개 (공개 데이터 스크래핑)
      * 위시켓 예시: "네이버 뉴스 스크래핑", "쿠팡 가격 모니터링"

  Day 4-7: Upwork 네트워크 구축
    - Upwork에서 신뢰할 만한 개발자 5-10명 컨택
    - 기준:
      * Top Rated (상위 10%)
      * 웹 스크래핑 전문 (100+ 프로젝트)
      * 평점 4.8+ / 5.0
      * 시급 $15-30 (품질 vs. 가격)
    - 작은 테스트 프로젝트 ($50-100) 진행 → 품질 검증

Week 2: 샘플 프로젝트
  - 소규모 프로젝트 1개 완수 (포트폴리오용)
  - 자체 비용: $100-200
  - 위시켓에 성공 사례로 등록
```

### Week 3-4: 첫 프로젝트 수주 및 실행
```yaml
Week 3: 프로젝트 수주
  Day 15-18: 위시켓에서 스크래핑 프로젝트 찾기
    - 타겟 예산: ₩2,000,000 - 5,000,000 (작게 시작)
    - 타겟 유형:
      * 전자상거래 가격 모니터링
      * 뉴스/블로그 콘텐츠 수집
      * 경쟁사 제품 정보 수집
    - 제안서 작성:
      * 기술적 접근 방법 명시
      * 샘플 프로젝트 포트폴리오
      * 빠른 납기 강조 (2주)

  Day 19-21: 수주 성공 시 Upwork 외주
    - 프로젝트 요구사항 영문 번역
    - Upwork에 Job Posting (Private, 사전 컨택한 개발자만)
    - 예산: $200-500
    - 에스크로 설정 (마일스톤 기반 지급)

Week 4: 프로젝트 관리 및 납품
  Day 22-26: 진행 관리
    - 매일 Upwork 개발자와 진행 체크
    - 한국 클라이언트에 주 2회 진행 보고
    - 중간 검수 (50% 완료 시)

  Day 27-28: 최종 납품
    - Upwork 개발자로부터 최종 결과물 수령
    - 한국 클라이언트에 맞게 문서화 (한글)
    - 위시켓 납품 및 검수 완료
    - 결제 수령
```

## 차별화 전략

### 1. 빠른 납기
```
경쟁자: 2-4주 개발
나: 1-2주 (해외 개발자 시차 활용)

방법:
- 한국 낮 = 미국/유럽 밤 (24시간 작업 가능)
- 긴급 프로젝트 +20% 추가 요금
```

### 2. 품질 보증
```
리스크: 해외 개발자 품질 편차
해결:
- Upwork Top Rated만 사용 (상위 10%)
- 마일스톤 기반 지급 (50% → 100%)
- 1개월 무상 버그 수정 보증

신뢰 구축:
- "Upwork Top 10% 개발자 네트워크 보유" (마케팅)
```

### 3. 한국 사이트 전문성
```
해외 개발자 약점: 한국 사이트 구조 모름
나의 강점:
- 네이버, 쿠팡, 지마켓 등 사전 분석
- 한국 사이트 특화 가이드 제공 (영문)
- 한글 인코딩 이슈 사전 해결

차별화: "한국 사이트 스크래핑 전문"
```

### 4. 투명한 커뮤니케이션
```
클라이언트 불안 제거:
- 주 2-3회 진행 보고 (스크린샷 포함)
- Notion 프로젝트 대시보드 공유
- 언제든 질문 가능 (24시간 내 답변)

고객 만족도 ↑ → 재의뢰 및 추천
```

## 리스크 및 완화 전략

### 리스크 #1: 품질 및 납기 지연
```
문제: 해외 개발자 품질 낮거나 지연
영향: 클라이언트 클레임, 위시켓 평점 하락

완화:
1. Top Rated만 사용 (상위 10%)
2. 테스트 프로젝트로 사전 검증
3. 마일스톤 기반 지급 (50% → 100%)
4. 버퍼 시간 확보 (2주 프로젝트 = 1주만 약속)
5. 백업 개발자 2-3명 대기

극단적 경우:
- 직접 개발 (시간 투자 ↑, 마진 유지)
- 다른 Upwork 개발자에게 긴급 외주
```

### 리스크 #2: 법적 이슈 (이용약관 위반)
```
문제: 웹 스크래핑 = 일부 사이트 이용약관 위반
영향: 법적 분쟁, 프로젝트 취소

완화:
1. 클라이언트에 명확히 고지:
   "타겟 사이트 이용약관 확인 필요"
2. 계약서에 면책 조항:
   "법적 책임은 클라이언트가 부담"
3. 공개 데이터만 수집 (로그인 필요 데이터 X)
4. robots.txt 준수 (Rate limiting)

거절 기준:
- 금융 데이터 (주식, 계좌)
- 개인정보 (이메일, 전화번호)
- 유료 콘텐츠 (뉴스 유료 기사)
```

### 리스크 #3: 한국어 소통 실수
```
문제: 클라이언트 요구사항 ↔ 영문 번역 오류
영향: 개발 방향 틀림, 재작업 필요

완화:
1. 상세 요구사항 문서화 (Notion)
2. 클라이언트 확인 (3번):
   - 요구사항 정리 후 확인
   - 개발 50% 시 중간 확인
   - 최종 납품 전 검수
3. ChatGPT로 번역 품질 검증
4. Upwork 개발자에게 스크린샷/예시 제공
```

### 리스크 #4: 경쟁 및 가격 하락
```
문제: 다른 중개자 등장, 가격 경쟁
영향: 마진 감소

완화:
1. 품질 차별화 (Top Rated 개발자)
2. 빠른 납기 (시차 활용)
3. 한국 사이트 전문성 (진입 장벽)
4. 고객 관계 구축 (재의뢰 유도)
5. 틈새 시장: "네이버 스크래핑 전문"
```

## 확장 경로 (Month 2-6)

### Month 2-3: 프로세스 최적화
```
목표: 시간당 수익 증가

최적화:
- 표준 요구사항 템플릿 (한글 → 영문)
- Upwork 개발자 전담 팀 (3-5명)
- 프로젝트 유형별 가이드 (전자상거래, 뉴스 등)

결과:
- 프로젝트 관리 시간 50% 감소
- 월 3-5개 프로젝트 동시 진행 가능
- 월 수익: ₩6,000,000 - 12,000,000
```

### Month 4-6: 자체 개발팀 구축
```
목표: Upwork 의존도 낮추기

전략:
- 베트남/인도 개발자 직접 채용 ($800-1,200/월)
- 장점:
  * Upwork 수수료 절약 (10%)
  * 커뮤니케이션 개선
  * 품질 일관성
- 단점:
  * 관리 복잡도 ↑
  * 고정 비용 발생

계산:
- 개발자 2명: $2,000/월 (₩2,680,000)
- 월 10개 프로젝트 처리 가능
- 수주 ₩30,000,000 - 외주 ₩2,680,000 = 마진 ₩27,320,000 (91%)
```

### Month 6+: 플랫폼화
```
목표: 위시켓 의존도 낮추기

자체 플랫폼:
- "한국 웹 스크래핑 전문 업체" 브랜드
- 웹사이트 + SEO ("네이버 스크래핑", "쿠팡 데이터")
- 기업 직접 영업 (위시켓 수수료 20% 절약)

예상:
- 월 수익: ₩40,000,000+
- 자체 고객 비율: 50%+
```
