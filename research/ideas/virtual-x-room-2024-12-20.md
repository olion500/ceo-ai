# Virtual X Room - 커플 추억 3D 가상공간

## 아이디어 요약

커플의 사진을 업로드하면 자동으로 3D 가상 공간을 생성하고, AI가 사진에서 위치/시간/인물 데이터를 추출하여 정리된 추억 공간을 만들어주는 서비스. 사용자는 가상 공간을 돌아다니며 추억을 회상할 수 있고, AI가 자동으로 영상도 생성해준다.

**핵심 기능:**
- 사진 업로드 → 자동 3D 가상공간 생성
- AI 메타데이터 추출 (위치, 시간, 인물)
- 가상공간 탐험 인터페이스
- 자동 추억 영상 생성

## 기술 스택 제안

### MVP 기준

#### Frontend
- **3D 엔진:** Three.js (WebGL 기반, VR 헤드셋 불필요)
- **프레임워크:** React 18 + TypeScript
- **UI:** MUI v7 (빠른 프로토타이핑)
- **상태관리:** TanStack Query (서버 상태)

#### Backend
- **런타임:** Node.js + Express
- **DB:** PostgreSQL (메타데이터) + S3 (사진 저장)
- **AI:** Google Vision API (얼굴/장소 인식) + OpenAI (자동 캡션)
- **영상 생성:** FFmpeg (서버 사이드) 또는 Remotion (React 기반)

#### 인프라
- **호스팅:** Vercel (프론트) + Render (백엔드)
- **CDN:** Cloudflare (사진 로딩 최적화)
- **분석:** Mixpanel (유저 행동) + Sentry (에러 추적)

### 기술적 도전 과제

1. **3D 렌더링 성능**
   - 문제: 사진 100장 → 3D 공간 로딩 느림
   - 해결: Lazy loading, LOD (Level of Detail), 웹워커

2. **메타데이터 정확도**
   - 문제: AI가 얼굴/장소 오인식
   - 해결: 수동 편집 기능 + 피드백 학습

3. **영상 생성 시간**
   - 문제: 서버 사이드 렌더링 느림
   - 해결: 비동기 작업 큐 (Bull/Redis) + 진행 상황 알림

4. **스토리지 비용**
   - 문제: 사진 원본 저장 비용 증가
   - 해결: 압축 (WebP) + 무료 티어 100장 제한
