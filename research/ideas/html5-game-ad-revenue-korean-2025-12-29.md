# 한국 문화 HTML5 게임 + 광고 수익

## 한 줄 요약

초성 퀴즈, 한국사 퀴즈 등 한국 문화 기반 무료 HTML5 게임. 광고로 수익화.

## 비즈니스 모델

**타입:** Ad-Supported Free Game
**수익원:** Google AdSense (95%)
**플랫폼:** 웹 (itch.io, 자체 도메인)
**타겟:** 한국 사용자 (학생, 직장인)

## 검증된 수요

```
Online Solitaire:
- $10,000/월 (광고 95%)
- 트래픽: 수십만 플레이/월

한국 시장:
- "무료 게임" 검색: 월 50,000+
- 웹 게임 인기 (접근성 ↑)
```

## 게임 아이디어

### #1: 초성 퀴즈 게임
```
게임플레이:
- 초성 제시 (ㅅㅅㅅ)
- 4개 선택지 (사과, 삼성, 손흥민, 설날)
- 시간 제한 (10초)

난이도:
- Easy: 3글자 초성
- Hard: 5-6글자 초성

소셜:
- 점수 카카오톡 공유
- 친구 도전
```

### #2: 한국사 퀴즈
```
게임플레이:
- 역사 질문 + 4지선다
- 레벨별 난이도 (삼국시대 → 현대)
- 해설 제공 (교육적)

타겟:
- 학생 (수능 준비)
- 직장인 (상식 테스트)
```

### #3: 한국 지명 퀴즈
```
게임플레이:
- 지도에서 지명 클릭
- 서울 25구, 광역시, 도
- 시간 제한

소셜:
- "나는 지리 박사" 공유
```

## 실행 계획 (1주)

### Ship in 1 Week
```yaml
Day 1-2: ChatGPT로 게임 코드 생성
  - Phaser.js 또는 Vanilla JS
  - 초성 퀴즈 로직
  - 스코어보드

Day 3-4: 광고 통합
  - Google AdSense 신청
  - 광고 배치 (게임 오버 시)

Day 5: 디자인
  - 한국 느낌 (한글 폰트)
  - 색상 (태극기 색)

Day 6-7: 배포
  - itch.io 무료 배포
  - 또는 Vercel (자체 도메인)
```
