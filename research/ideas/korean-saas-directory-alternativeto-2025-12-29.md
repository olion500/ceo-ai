# 한국 SaaS 대안 찾기 디렉토리

## 한 줄 요약

AlternativeTo.net 스타일의 한국어 SaaS 디렉토리. 한국 사용자가 해외 SaaS 대안을 쉽게 찾을 수 있는 플랫폼.

## 비즈니스 모델

**타입:** Platform + Listing + Ad Revenue + Affiliate
**수익원:**
1. SaaS 업체 프리미엄 리스팅 (₩50,000-200,000/월)
2. Google AdSense (트래픽 10K+/월 시)
3. Affiliate 수수료 (SaaS 가입 추천)

## 실행 계획 (1주)

### Ship in 1 Week 패턴 적용
```yaml
Day 1-2: MVP 개발
  - Next.js 14 + Supabase (무료)
  - 기능:
    * SaaS 검색 (이름, 카테고리)
    * 대안 리스팅 (Notion → Asana → Trello)
    * 비교 테이블
  - 디자인: Tailwind UI (빠른 구현)

Day 3-4: 콘텐츠 제작
  - 한국 SaaS 100개 수동 등록:
    * 노션, 슬랙, 피그마, 지라, 트렐로 등
    * 각 SaaS: 설명, 가격, 장단점
  - SEO 최적화:
    * 메타 태그: "노션 대안", "슬랙 대안"
    * 각 페이지 독립 URL

Day 5-7: 런칭 및 홍보
  - Vercel 배포 (무료)
  - Show HN 런칭
  - Reddit r/Korea, r/SaaS 홍보
  - 한국 커뮤니티 (Okky, GeekNews)
```

## 차별화 전략

### 1. 한국어 100%
```
AlternativeTo.net: 영문, 한국 SaaS 약함
나: 한국어 전용, 한국 사용자 타겟

검색어 예시:
- "노션 대안" → Craft, Obsidian, Roam Research
- "슬랙 대안" → Discord, Microsoft Teams, 잔디
- "피그마 대안" → Sketch, Adobe XD, Penpot
```

### 2. 한국 특화 필터
```
독특한 필터:
- "한국어 지원" (공식 한글 UI)
- "국내 결제" (원화, 신용카드)
- "한국 고객센터" (한국어 지원)
- "무료 플랜" (한국 스타트업 친화)
```

### 3. 실사용 리뷰
```
기능: 사용자 리뷰 시스템
- 한국 사용자 관점 리뷰
- 장단점 (한국 맥락)
- 가격 대비 가치

예시: "Notion 한국어 검색 약함" (실제 불만)
```

## 예상 수익

### Month 1-2: 트래픽 구축
```
목표: SEO 기반 유기적 트래픽
- Google 검색: "노션 대안" 상위 노출
- 일 방문자: 100-500명
- 수익: ₩0-100,000 (초기 제휴)
```

### Month 3-6: 수익화
```
프리미엄 리스팅:
- SaaS 업체 10개 × ₩50,000/월 = ₩500,000

Google AdSense:
- 월 트래픽 10,000명
- CTR 2%, CPC ₩300
- 수익: ₩60,000/월

Affiliate:
- 월 가입 20명 × ₩10,000 = ₩200,000

총: ₩760,000/월 (3개월 후)
```

### Month 12: 확장
```
프리미엄 리스팅: 30개 × ₩100,000 = ₩3,000,000
AdSense: ₩500,000
Affiliate: ₩1,000,000
기업 스폰서 배너: ₩500,000

총: ₩5,000,000/월
```

## 리스크 및 완화

**리스크 #1: SEO 시간 필요**
- 완화: Show HN으로 초기 트래픽
- 완화: Reddit/커뮤니티 홍보
- 예상: 3-6개월 SEO 성과

**리스크 #2: 콘텐츠 업데이트 부담**
- 완화: 사용자 생성 콘텐츠 (리뷰)
- 완화: SaaS 업체 직접 업데이트 (프리미엄 회원)
